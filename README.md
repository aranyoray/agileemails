# AgileEmails - Smart Email Management Chrome Extension

**Repository:** [https://github.com/minicompetitioncom-rgbf/AgileEmails](https://github.com/minicompetitioncom-rgbf/AgileEmails)

## Overview
AgileEmails is an intelligent Chrome extension designed to help you manage your Gmail inbox more efficiently. It automatically categorizes emails, assigns priority scores, provides DND mode, and helps you focus on what matters.

## Features

### Core Features
- **Automatic Email Categorization**: Emails are automatically sorted into categories:
  - Academics
  - Work
  - Jobs
  - Finance
  - Personal
  - Auth
  - Newsletter
  - Others

- **Priority Scoring (1-5)**: 
  - 5: Urgent (Red)
  - 4: High (Orange)
  - 3: Medium-High (Yellow)
  - 2: Medium (Light Green)
  - 1: Low (Dark Green)

- **DND Mode**: Configure rules to hide or silence emails during specific times or from specific senders, with exceptions for urgent keywords and deadlines.

- **Auto-Delete Rules**: Automatically delete emails based on category and age (1 day, 7 days, 30 days, 3 months, 6 months, 1 year).

- **Reply Queue**: Organized view of emails that need replies, sorted by importance.

- **Newsletter Detection**: Automatically identifies newsletters and mass-emails vs personalized emails.

- **Important Info Extraction**: Automatically extracts and displays:
  - Important links
  - Dates mentioned
  - Money amounts
  - Tasks/action items

- **Thread Summarization**: Summary popup for email threads showing key information.

## Installation

1. Download the extension ZIP file
2. Extract it to a folder
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" (toggle in top right)
5. Click "Load unpacked" and select the extracted folder
6. The extension will be installed and ready to use

## Usage

1. **Access the Extension**: Click the AgileEmails icon in your Chrome toolbar
2. **View Your Queue**: Use the tabs to navigate between:
   - Reply Queue (emails needing responses)
   - Missed (important emails from previous days)
   - Today (today's emails sorted by importance)
   - Non-Important (low priority emails)

3. **Configure Settings**: Click the settings button to:
   - Choose your pricing tier
   - Customize categories and colors
   - Set up DND rules
   - Configure auto-delete settings
   - Adjust general preferences

4. **Use DND Mode**: Configure rules to automatically hide emails during specific times or from specific senders. Exceptions can be set for urgent emails, deadlines, or specific keywords.

## Gmail Integration

The extension automatically integrates with Gmail and will:
- Show priority color borders on email threads
- Display category badges
- Hide emails when DND mode is active
- Extract and display important information from threads

## Privacy

- All email processing happens locally in your browser
- No data is sent to external servers
- Email data is stored only in Chrome's local storage
- Full privacy policy available in settings
